Instructions

Feel free to post your ideas for TODOs here. If it seems more like a bug in the existing code, post it to bugs instead. Useful commands that can be put inside the TODOs include

  • Link items to done when done by inserting 
    [[!tag  done]]
  • Link items to appropriate owner when assigning a todo by inserting, for example, 
    [[!tag  owner/arlolra]]
  • Link items to appropriate priority levels (high, medium, low) by inserting, for example, 
    [[!tag  priority/medium]]
    (note: by default, TODOs that don't have a priority tag are classified as low)
